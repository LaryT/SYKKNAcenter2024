GameCenter

for basketball

είναι ένα web application, μια διαδικτυακή εφαρμογή, που διαχειρίζεται με απλό τρόπο, αλλά και ταυτόχρονα έχει την δυνατότητα να παρακολουθεί και να παρουσιάζει στο web, εθνικά - τοπικά πρωταθλήματα μπάσκετ, τουρνουά που διοργανώνουν σύλλογοι μπάσκετ, οργανώνοντας τις κατηγορίες, ομίλους, βαθμολογίες και ρόστερ.

Πρόκειται για ένα web app απόλυτα συμβατό σε υπολογιστές και κινητές συσκευές

Πως λειτουργεί

Σύστημα διαχείρισης

Χρήσιμο για Ενώσεις, Οργανισμούς, Συλλόγους ή μεμονωμένα άτομα που επιθυμούν να διοργανώσουν ένα τουρνουά - πρωτάθλημα και δεν διαθέτουν την υποδομή ώστε να δημιουργήσουν ηλεκτρονικά κατηγορίες, ομίλους, ομάδες που μετέχουν, ρόστερ συλλόγων.

ΔΕΝ απαιτείται υποδομή σε DataBase
Web application

Ιδανικό για όσους θέλουν, να ενσωματώσουν το GameCenter στο site τους, με σκοπό να παρουσιάζουν διοργανώσεις ή πρωταθλήματα άμεσα και χωρίς κόπο, προβάλλοντας αυτόματα όλες τις εξελίξεις των αποτελεσμάτων, των βαθμολογιών μέσω της εφαρμογής.

Πρακτικό

Γιατί δεν χρειάζεται εσύ (ως κάτοχος site), να συντηρείς και να αφιερώνεις χρόνο αναζήτησης και εισαγωγής στοιχείων. Απλά ενσωματώνεις το web app στο site σου. Η τοποθέτηση του GameCenter app στο site σου, γίνεται πανεύκολα στο σημείο που θέλεις, με την αντιγραφή και επικόλληση μια γραμμής κώδικα (ενσωμάτωση HTML). Ο κώδικας που χρειάζεσαι παρέχεται αυτόματα από την ίδια την εφαρμογή, ανάλογα με ποιο τμήμα θέλεις να συμπεριλάβεις στο site σου

Συμβατότητα

Το GameCenter app είναι δομημένο και αναπτυγμένο με κύριο σκοπό την όσο το δυνατό μεγαλύτερη συμβατότητα σε συσκευές. Τρέχει σ' όλους τους δημοφιλής φυλλομετρητές browsers, chrome, safari, firefox, ie κλπ, με συμβατότητα προς τις παλιότερες εκδόσεις (σε ποσοστό 98,6%), τόσο σε υπολογιστές όσο και σε κινητά smartphones τηλέφωνα.

Ευκολόχρηστο

Η εισαγωγή στοιχείων από την πλευρά των υπευθύνων διοργανώσεων είναι πανεύκολη μια και απαιτεί βασικές γνώσεις χειρισμού υπολογιστικών φύλλων (τύπου excel). Η ενσωμάτωση σε site γίνεται με την αντιγραφή μίας μόνο σειράς κώδικα, μια διαδικασία συνηθισμένη και απλή για κάθε κάτοχο site. Η χρήση από την πλευρά των αναγνωστών απλή. Ευανάγνωστη και εύκολη πρόσβαση σε όλα τα επίπεδα των πληροφοριών.
